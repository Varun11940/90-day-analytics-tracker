# 90-Day Analytics Engineer Tracker

A personal progress tracker for my 90-day journey to become an Analytics Engineer.

**Live site:** https://varun11940.github.io/90-day-analytics-tracker/

## What's inside

- ✅ All 90 days with specific daily tasks (SQL, dbt, LeetCode, DataLemur)
- 💾 Progress auto-saves to browser localStorage
- 📤 Export / import backup as JSON (so you never lose progress)
- 📊 Live stats: days done, tasks done, % complete

## Stack

Plain HTML + CSS + JS. No frameworks, no build step. Just open and go.

## How to use locally

```bash
git clone https://github.com/Varun11940/90-day-analytics-tracker.git
cd 90-day-analytics-tracker
# open index.html in your browser
```
